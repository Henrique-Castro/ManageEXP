# Componentes

Os componentes tem como objetivos principais diminuir a repetição de código, facilitar o controle da estilização do sistema e deixá-lo escalável. Você pode ler um pouco mais sobre [nesse link](https://reactjs.org/docs/components-and-props.html).

Abaixo seguem os componentes utilizados na plataforma e suas especificações:

## Div

O componente [Div]() tem como obejtivo facilitar a criação de blocos e caixas no frontend, sem a necessidade de passarmos estilizações por css ou a adição de classes para estilização
