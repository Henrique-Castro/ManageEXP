# Seja bem vindo
Seja bem vindo à documentação do projeto ManageEXP!

A plataforma ManageEXP tem o objetivo de facilitar o controle de dados através da integração com o [Zabbix](https://www.zabbix.com/) de forma inteligente, rápida e simples!
